# Docs Index

Документация разложена по доменам:

- `docs/services/mapper/` - пользовательская и разработческая документация по CLI-инструменту.
- `docs/architecture/` - архитектурная карта, ADR и агентные архитектурные каноны.
- `docs/contracts/` - контрактные правила интерпретации, рендер-контракты, test policy.
- `docs/product-ux/` - продуктовые спецификации и решения уровня продукта.
- `docs/integrations/` - интеграционные документы (по конкретным внешним системам).
- `docs/reports/YYYY-MM-DD/` - отчеты и аудиты по датам.
- `docs/_index/redirects.md` - таблица переездов старых путей в новые.
